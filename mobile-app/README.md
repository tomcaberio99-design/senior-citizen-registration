# Senior Citizen Mobile App

Expo / React Native starter for the MSWDO Alegria Senior Citizen Services Portal.

## Run

1. `cd mobile-app`
2. `npm install`
3. `npm start`

## Included Screens

- Home
- Register
- Login
- Dashboard
- Payout
- Profile

## Notes

- The current app uses local demo state for registration and login flow.
- Branding assets are copied from the web app `public/` folder.
